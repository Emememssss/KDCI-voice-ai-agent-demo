// Demo Instructions

How to Try This

Two ways to interact — choose based on what you want to experience.

📞 Via Phone Call

Full end-to-end experience including live human transfer.

  1. 01 — Dial +1 681 293 1947 (hold 0 for + on your mobile)
  2. 02 — Emman (AI) answers. Ask 2–3 questions from the list below.
  3. 03 — Say "I want to talk to a human" or "connect me to a person"
  4. 04 — Wait — the call transfers automatically to a live KDCI agent
  5. 05 — You're now speaking directly with a human representative

Note: International call rates apply from Philippines (~₱15–25/min via Smart/Globe). Use WiFi calling or Google Voice to reduce cost.

🌐 Via Browser Widget

Quick demo of AI conversation — no phone needed.

  1. 01 — Click "Start a call" in the widget above
  2. 02 — Allow microphone access when prompted
  3. 03 — Speak naturally — ask Emman questions
  4. 04 — Say "I want to talk to a human" to see escalation behavior
  5. 05 — Note: transfer to live agent is not available in browser mode

Limitation: Browser widget cannot forward to a live phone agent. This is an ElevenLabs platform constraint for web-based sessions. Full transfer only works via phone call.
